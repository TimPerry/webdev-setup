# Web dev setup

## Install

```
git clone git@github.com:TimPerry/webdev-setup.git
cd webdev-setup
./install.sh
```

## What you get
- homebrew
- ansible
- virtualbox
- dockertoolbox
- vagrant
- vagrant-vbguest plugin
- google-chrome
- firefox
- 1password
- atom
- intellij-idea
- caffeine
- ccleaner
- iterm2
- slack
- screenhero
- zsh
- pure prompt (zshrc theme)
